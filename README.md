# Resume-Parsing-Candidate-Ranking
Automated NLP-based system to parse PDF resumes and rank candidates using TF-IDF and cosine similarity. Extracts job description, computes relevance scores, and outputs ranked results in CSV, improving recruitment efficiency and scalability.
# 📄 Resume Parsing & Candidate Ranking System

## 🚀 Overview
This project is an NLP-based automated resume screening system that ranks candidates based on job description relevance.

## 🎯 Features
- Extracts text from PDF resumes
- Processes job descriptions
- TF-IDF vectorization
- Cosine similarity scoring
- Candidate ranking
- CSV export

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- pdfminer.six

## ⚙️ Workflow
1. Upload resumes (PDF)
2. Provide job description
3. Text extraction
4. TF-IDF transformation
5. Similarity computation
6. Ranking generation
