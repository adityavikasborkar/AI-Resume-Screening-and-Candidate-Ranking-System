# AI Resume Screening & Candidate Ranking System
Automated NLP-based system to parse PDF resumes and rank candidates using TF-IDF and cosine similarity. Extracts job description, computes relevance scores, and outputs ranked results in CSV, improving recruitment efficiency and scalability.

## Problem Statement
Recruiters often receive hundreds of resumes for a single job opening. 
Manually screening candidates is time-consuming and inefficient.

This project automates resume parsing and ranks candidates based on job description relevance using NLP techniques.

---

## Objectives
- Extract key information from resumes
- Match resumes with job descriptions
- Rank candidates based on similarity score

---

## System Architecture
1. Resume Input (PDF/DOCX)
2. Text Extraction
3. NLP Processing
4. Feature Extraction (TF-IDF / embeddings)
5. Similarity Calculation
6. Candidate Ranking

---

## Technologies Used
- Python
- NLP (NLTK / spaCy / TF-IDF)
- Scikit-learn
- Flask (optional for UI)

---

## Features
- Extracts:
  - Name
  - Email
  - Skills
  - Experience
- Calculates similarity score
- Ranks candidates automatically

---

## Output
- Ranked list of candidates
- Match percentage score
